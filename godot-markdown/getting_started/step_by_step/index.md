# Step by step

This series builds upon the [Introduction to Godot](../introduction/index.md#toc-learn-introduction) and will get you started with the editor and the
engine. You will learn more about nodes and scenes, code your first classes with
GDScript, use signals to make nodes communicate with one another, and more.

The following lessons are here to prepare you for [Your first 2D game](../first_2d_game/index.md#doc-your-first-2d-game), a
step-by-step tutorial where you will code a game from scratch. By the end of it,
you will have the necessary foundations to explore more features in other
sections. We also included links to pages that cover a given topic in-depth
where appropriate.

* [Nodes and Scenes](nodes_and_scenes.md)
* [Creating instances](instancing.md)
* [Scripting languages](scripting_languages.md)
* [Creating your first script](scripting_first_script.md)
* [Listening to player input](scripting_player_input.md)
* [Using signals](signals.md)
