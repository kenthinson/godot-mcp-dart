# Compiling for visionOS

#### SEE ALSO
This page describes how to compile visionOS export template binaries from source.
If you're looking to export your project to visionOS instead, see [Exporting for visionOS](../../../tutorials/export/exporting_for_visionos.md#doc-exporting-for-visionos).

Compiling instructions for visionOS are currently identical to
[Compiling for iOS](compiling_for_ios.md#doc-compiling-for-ios), except you should replace instances of
`platform=ios` with `platform=visionos` in the SCons options.
See the linked page for details.
