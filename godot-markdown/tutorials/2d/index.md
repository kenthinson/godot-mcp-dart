Godot includes a dedicated 2D renderer and 2D physics engine, as well as
2D-specific features like tilemaps, particles, and animation systems. This
section covers most 2D-specific topics in Godot.

For 2D topics not covered in this section, see also [2D skeletons](../animation/2d_skeletons.md#doc-2d-skeletons) and
[2D navigation overview](../navigation/navigation_introduction_2d.md#doc-navigation-overview-2d). For using physics in 2D, see
[Physics](../physics/index.md#doc-physics-index). There is also a step-by-step tutorial on creating a 2D
game in [Your first 2D game](../../getting_started/first_2d_game/index.md#doc-your-first-2d-game).
